# Assignments: Cannot save advanced assignment change from a report

When a user attempts to make an advanced assignment change from a report, the change does not save, and the user sees the following message:

"422: APIModel INTERNAL does not support field removeOldAssignments"

**Workaround**

You can change advanced assignments from the project's task list.

_First reported on February 4, 2026._

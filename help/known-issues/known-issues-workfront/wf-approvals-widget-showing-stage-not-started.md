# Approvals: My Approvals widget displays proofs that are not in the stage the user is assigned to

When a user views their My Approvals widget in Home, they see proofs where the user is assigned to a stage, but the proof is not in the stage that the user is assigned to. This may make it difficult for the user to quickly see what action is necessary. 

_First reported on February 16, 2026._

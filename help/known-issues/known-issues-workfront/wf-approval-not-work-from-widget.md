# Approvals: Cannot approve document from My Approvals widget in Home

When a user attempts to approve a document from the My Approvals widget in Home, the document disappears from the widget. However, the document is not approved, and if the user refreshes the page, the document reappears in the widget.

_First reported on February 10, 2026._

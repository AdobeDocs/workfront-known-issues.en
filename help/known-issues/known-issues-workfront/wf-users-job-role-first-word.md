# Users: Can only search job roles by first word

>[!NOTE]
>
>This issue was fixed on November 17, 2025.

When an admin is creating or editing a user and searches job roles, the search returns only job roles beginning with the search term, not all job roles that include the search term.

_First reported on October 21, 2025._
